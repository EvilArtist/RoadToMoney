**Road to Money** là game lặn biển casual 3D first-person xây dựng trên Godot 4.3 Mono với C#.

Player lặn xuống đại dương, bắt hải sản bằng phím E, nổi lên bán cho shop để kiếm coins mua upgrade túi đựng, bình oxy, đèn lặn và công cụ bắt.

Môi trường có seabed nghiêng dần theo độ sâu, san hô, rong biển, đá ngầm, cùng các loài cá như Clownfish, Angelfish, Yellow Tang bơi thành đàn theo thuật toán Boids.

Hệ thống core đã hoàn thiện gồm 6 Autoload singletons, resource system với 8 loài hải sản, creature AI với FSM, shop UI, upgrade system và save/load.

Còn lại cần làm: Main Menu, Settings, performance optimization, và polish trước khi release.
