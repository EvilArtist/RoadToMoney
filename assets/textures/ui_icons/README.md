# UI Icon Assets

Thư mục này chứa các icon PNG cho BottomIconBar trong HUD.

## Danh sách file cần upload:

| File | Mô tả | Dùng cho |
|------|--------|----------|
| `icon_shop.png` | Icon cửa hàng | Nút Shop |
| `icon_upgrade.png` | Icon nâng cấp | Nút Upgrade |
| `icon_book.png` | Icon sổ tay / field guide | Nút Book |
| `icon_menu.png` | Icon menu / hamburger | Nút Menu chính |

## Kích thước đề xuất:
- 64×64 px hoặc 128×128 px
- Định dạng PNG với nền trong suốt (alpha)
- Sau khi upload, Godot sẽ tự import khi mở project
